# Data Analyst Assignment Portfolio

## Project Overview
This repository contains solutions for a multi-disciplinary technical assessment covering SQL database management, spreadsheet data manipulation, and Python programming logic.

## Logic and Assumptions
- **SQL:** Used `DENSE_RANK()` for finding the second-highest values to ensure that tied results are handled correctly.
- **Excel:** Used the `INT()` function to extract dates from timestamps. This is critical because Excel treats dates as integers and times as fractional parts of a day.
- **Python:** Employed floor division (`//`) and modulo (`%`) operators for time conversion to ensure integer outputs.
